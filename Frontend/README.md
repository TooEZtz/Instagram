# Frontend Folder

This folder contains all frontend files for the Instagram Clone project.

## Structure

- `html/` - All HTML files
- `css/` - All CSS/stylesheet files
- `js/` - All JavaScript files

## File Organization Rules

**IMPORTANT:** Files must be placed in their respective folders:
- ✅ HTML files → `html/` folder
- ✅ CSS files → `css/` folder
- ✅ JavaScript files → `js/` folder

## Usage

All HTML, CSS, and JavaScript files for the frontend should be stored in their designated folders.
